KDM CLI Tool

A command-line utility for working with PDFs and images. Includes commands to convert PDF pages to images and merge images into a PDF.

Requirements

  • Python 3.8 or higher
  • Poppler (for PDF to image conversion)

Installation

macOS & Linux

  1. Install Python: 
    brew install python
  2. Install Poppler: 
    brew install poppler
  3. Clone the repo and install: 
    
git clone https://github.com/kudadonbe/tools.git
cd tools
pip3 install --user -e .[dev]

Windows

  1. Install Python from python.org, check "Add Python to PATH".
  2. Download and extract Poppler from GitHub, then add its /bin folder to PATH.
  3. Clone and install: 
    
git clone https://github.com/kudadonbe/tools.git
cd tools
pip install -e .[dev]

Usage

Once installed, run:

kdm --help

Common commands:

  • kdm pdf2images – Convert PDF to image files.
  • kdm images2pdf – Merge multiple images into a PDF.

Development

  1. Modify source files directly after editable install (pip install -e .[dev]).
  2. Use kdm commands to test.
  3. Commit and push changes: 
    
git add .
git commit -m "Update"
git push origin main
